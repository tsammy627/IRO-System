# IRO (Issues Reporting Outlet) System

## 📘 Description
The IRO system is a web-based application that allows students and indigenes to report issues and track their resolution.

## 🎯 Features
- User Registration and Login
- Issue Reporting
- Issue Tracking
- Admin Management
